IZipKit.Count

IZipKit

object.Count
The object is always an IZipKit.

Returns the number of items available from the Item property.

Caution: Count returns the number of items in the memory representation of the archive. If AutoSave is False, and items have been added to or deleted from the archive, the number of items in
the memory representation can be different then the number of items in the disk archive.

See also: AutoSave
