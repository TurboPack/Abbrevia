TAbBaseBrowser.ArchiveProgressMeter

TAbBaseBrowser

property ArchiveProgressMeter : TAbMeter

Specifies a synchronized meter for the OnArchiveProgress event.

The ArchiveProgressMeter property provides a mechanism to associate a TAbMeter with the OnArchiveProgress event. If a meter is specified, it will be updated to reflect the progress given by the OnArchiveProgress event. This occurs even if no event handler has been assigned to the event.

See also: OnArchiveProgress
