TAbBaseBrowser.ItemProgressMeter

TAbBaseBrowser

property ItemProgressMeter : TAbMeter

Specifies a synchronized meter for the OnArchiveItemProgressEvent.

The ItemProgressMeter property provides a mechanism to associate a TAbMeter with the OnArchiveItemProgress event. If a meter is specified then it will be updated to reflect the progress given by the OnArchiveItemProgress event. This occurs even if no event handler has been assigned to the event.

See also: OnArchiveItemProgress
